NAME SYNTAX COMBINATOR DESCRIPTION
Descendant Combinator A B This combinator, materialized by a single space, targets an element B contained in an element A, whatever its degree of relationship: a child or a child of a child.
Direct Child Combinator A > B Unlike the previous one, this combinator, materialized by a superior, targets an element B which is the direct child of an element A.
Adjacent Sibling Combinator A+B This combinator, represented by a plus, targets an element B immediately preceded by an element A. The adjacent word is important: only the first element after element A will be targeted.
General Sibling Combinator A~B This combinator targets an element B preceded by an element A. Unlike the adjacent combinator where only the first brother is targeted, here, all brothers are concerned.

NOM SYNTAXE DESCRIPTION DU SELECTEUR
Combinateur descendant A B Ce combinateur, matérialisé par un simple espace, cible un élément B contenu dans un élément A, quelque soit son degré de parenté : un enfant ou un enfant d’enfant.
Combinateur enfant A > B Contrairement au précédent, ce combinateur, matérialisé par un supérieur, cible un élément B qui est l’enfant direct d’un élément A.
Combinateur adjacent A + B Ce combinateur, matérialisé par un plus, cible un élément B immédiatement précédé par un élément A. Le mot adjacent est important : seul le premier élément après l’élément A sera ciblé.
Combinateur général sibling A ~ B Ce combinateur cible un élément B précédé par un élément A. Contrairement au combinateur adjacent où seul le premier frère est ciblé, ici, tous les frères sont concernés.
